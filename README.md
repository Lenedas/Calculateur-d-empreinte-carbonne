# Calculateur-d-empreinte-carbonne
ceci est un calculateur d'empreinte Carbonne pour entreprise de logistique pour camion transporteurs, il calcule le poids des palettes et le nombre de kilomètre a parcourir et avec un facteur Carbonne fait une équation donnant le résultat et sa formule.
